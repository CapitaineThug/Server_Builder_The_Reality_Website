Qu'est-ce qu'Unreal Engine

En 1995, la première version d’Unreal Engine est créée. Son inventeur, Tim Sweeney, en aurait programmé 90% avec le langage C++. Mais qu’est-ce qu’Unreal Engine ? C’est un moteur graphique 3D permettant de créer des jeux en 2 ou 3D. La dernière version disponible actuellement (janvier 2023) est Unreal engine 5.1.

Programmation

Lors de la programmation de jeu dans l’éditeur, il est possible d’utiliser 2 langages de programmation :

C++

C’est un langage de programmation de haut niveau utilisé dans Unreal Engine.

Blueprint

Le Blueprint est un code qui peut être aisément utilisé directement dans l’éditeur. Il est entièrement graphique et facile d’accès.

Plateformes supportées

Unreal Engine 5.1 supporte actuellement ces différentes plateformes comme support de jeu:

  • Windows 10
  • PlayStation
  • Xbox
  • Nintendo Switch
  • MacOS
  • Android
  • IOS
  • Oculus
  • Linux
  • Steamdeck
